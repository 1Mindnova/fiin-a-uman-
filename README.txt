
Projekt: fiin-a-uman – Surah Al-Insan
Dieses Projekt synchronisiert Verse des Koran-Kapitels 'Der Mensch (Al-Insan)' mit einer Audioaufnahme.

Inhalt:
- index.html: Hauptseite
- style.css: Design und Schrift
- script.js: Steuerung der Anzeige synchron zur Audio
- verses.json: Zeitlich getaktete Versdaten
- audio.mp3: (NICHT ENTHALTEN – bitte separat hochladen)

Anleitung:
1. Alle Dateien auf GitHub Pages hochladen.
2. Sicherstellen, dass die Audio-Datei als 'audio.mp3' im selben Ordner liegt.
3. Webseite öffnen und auf ▶ Play klicken.
